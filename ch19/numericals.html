19.1 A particle called pion lives on average about 2.6 x 10-8 s when at rest in the laboratory. It then changes to another form. How long would such a particle live when shooting through the space at 0.95c?

Given

  1. Proper Time = to = 2.6 x 10-8 s
  2. Velocity = v = 0.95c
  3. Speed of Light = c = 3.0 x 108 ms-1

To Find

  1. Dilated Time = t = ?

Solution

  1. As,
  2. $$ t = \frac{t_o}{ \sqrt{1 - \frac{v^2}{c^2} } } $$
  3. $$ t = \frac{ 2.6 \times 10^{-8} }{ \sqrt{1 - \frac{(0.95c)^2}{c^2} } } $$
  4. $$ t = \frac{ 2.6 \times 10^{-8} }{ \sqrt{1 - \frac{0.9025c^2}{c^2} } } $$
  5. $$ t = \frac{ 2.6 \times 10^{-8} }{ \sqrt{1 - 0.9025 } } $$
  6. $$ t = \frac{ 2.6 \times 10^{-8} }{ \sqrt{ 0.0975 } } $$
  7. $$ t = \frac{ 2.6 \times 10^{-8} }{ 0.3122 } $$
  8. $$ t = 8.3 \times 10^{-8} s $$
